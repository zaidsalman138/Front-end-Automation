# Front-End UI Automation Framework

This repository contains a **frontend UI automation framework** built using **Selenium WebDriver (Java)** for validating modern web applications through real user workflows.

The framework follows **industry-standard automation practices** and is designed to be **scalable, maintainable, and CI/CD ready**.

---

## Key Features

- Selenium WebDriver automation using **Java**
- **Page Object Model (POM)** for clean and reusable test architecture
- **TestNG** for test execution and suite management
- **Cucumber (BDD)** for readable test scenarios
- **Maven** for dependency and build management
- **Data-driven testing** using external data sources
- Centralized logging using **Log4j**
- **Allure and Extent Reports** for detailed test reporting
- Support for listeners, utilities, and reusable components

---

## Tech Stack

- Selenium WebDriver (Java)
